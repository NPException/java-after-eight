Nice Java 8 code base that gets way nicer with Java 9-14.

### Java 9

#### Module system

* strong encapsulation:
	* in _genealogy_:
		* export `genealogist` and `articles`
	* in _genealogist_:
		* export nothing

	* update _genealogy_ and _genealogist_ accordingly

		</configuration>
